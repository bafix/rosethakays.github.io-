# 🚀 How to Put Rosetha's Portfolio Online (GitHub Pages)
## Step-by-step — no coding experience needed

---

## What you need
- A free GitHub account → https://github.com/signup
- The `index.html` file (already built)

---

## STEP 1 — Create a GitHub Account
1. Go to **https://github.com/signup**
2. Choose a username — ideally something like `rosethakays` or `dr-rosetha-kays`
3. Verify your email address

---

## STEP 2 — Create a New Repository
1. Once logged in, click the **"+"** icon (top right) → **"New repository"**
2. Name it exactly: `rosethakays.github.io`
   ⚠️ The name MUST follow this pattern: `yourusername.github.io`
   (replace "rosethakays" with whatever username you chose)
3. Set it to **Public**
4. Tick **"Add a README file"**
5. Click **"Create repository"**

---

## STEP 3 — Upload the Portfolio File
1. Inside your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop the `index.html` file onto the page
3. Scroll down, add a commit message like: `Add portfolio website`
4. Click **"Commit changes"**

---

## STEP 4 — Enable GitHub Pages
1. In your repository, click **"Settings"** (top menu)
2. Scroll down to **"Pages"** in the left sidebar
3. Under **"Source"**, select **"Deploy from a branch"**
4. Under **"Branch"**, select **"main"** and **"/ (root)"**
5. Click **"Save"**

---

## STEP 5 — Your Site is Live! 🎉
- Wait 1–2 minutes, then visit:
  **https://rosethakays.github.io**
  (replace with your actual username)
- GitHub will show you the live URL in the Pages settings section

---

## Optional: Custom Domain (e.g. rosethakays.com)
If you ever want a custom domain:
1. Buy a domain from Namecheap or Google Domains (~$10–15/year)
2. In GitHub Pages settings, enter your domain under "Custom domain"
3. Follow the DNS setup instructions GitHub provides

---

## Updating the Portfolio Later
Whenever you want to make changes:
1. Go to your repository on GitHub
2. Click on `index.html`
3. Click the ✏️ pencil (Edit) icon
4. Make your changes
5. Click "Commit changes"
The site updates automatically within a minute.
